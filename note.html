HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

!DOCTYPE html;
html lang="en"
head
title>My First Page</title>
</head>
<body>
  <h1>Hello World!</h1>
</body>
</html>
Note: HTML elements are defined by tags, written using angle brackets.

Page Structure

Every HTML page needs these basic structural elements:

  • <!DOCTYPE html> - Document type declaration
  • <html> - Root element
  • <head> - Meta information
  • <body> - Page content

Text Elements

HTML provides various elements for formatting text:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<p>Paragraph text</p>
<strong>Bold text</strong>
<em>Italic text</em>

Links and Images

Links and images are fundamental to web pages:

<a href="https://example.com">Click here</a>
<img src="image.jpg" alt="Description">

Lists

HTML supports ordered and unordered lists:

<ul>
  <li>Unordered item</li>
</ul>

<ol>
  <li>Ordered item</li>
</ol>

Tables

Tables are used to display data in rows and columns:

<table>
  <tr>
    <th>Header</th>
  </tr>
  <tr>
    <td>Data</td>
  </tr>
</table>

Forms

Forms allow users to input data:

<form>
input type="text" placeholder="Enter name
input type="email" placeholder="Enter email">
button type="submit">Submit</button>
/form
